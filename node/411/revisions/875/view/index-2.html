Problem accessing issues on Drupal.org since the Great Git Migration

The great git migration

So the great git migration is finally over! Well, almost over - for us there are still issues. The tags in our history were not properly converted and that required extra work from the already really busy migration team, so while they figure out a way to fix all this, the issues links are broken, as the provision and hostmaster nodes are unpublished.

This will hopefully be fixed within the coming days and we'll then make another announcement to let you know where the real repositories are.

In the mean time, the old repositories at git.aegirproject.org are still the authoritative repositories.
