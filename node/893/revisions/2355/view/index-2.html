1.4 released

The Aegir team is happy to release the fifth stable release in the 1.x branch. This release is to address a security vulnerability in the Eldir theme, part of hostmaster.

There are no other changes, but there are special instructions for people who use the Debian packages to install Aegir.

See the full release notes here: http://community.aegirproject.org/1.4
