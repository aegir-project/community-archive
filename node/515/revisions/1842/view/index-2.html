Hosting Platform Roles
Hosting Platform Roles is a Drupal module that extends Aegir to enable more control over what role(s) a client is assigned during site install. Since, by default, Aegir client's are given UID1 during site installs, this module also allows UID1 to be assigned to another user.

Hostmaster Services API integration aims at integration of the Services API framework into the Hostmaster suite of tools. It allows Aegir managers the ability to build Services API connections over any of the offered Servers for Services API.
