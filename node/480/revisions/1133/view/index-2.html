Aegir 1.0-rc2 is released!

Referenced Page: 
1.0-rc2 release notes

The Aegir team is very pleased to announce the second release candidate of Aegir 1.0!

Yes, you read that right: 1.0! We've decided to rename the 0.4 branch to 1.0.

This is the second Release Candidate from which we will test fixes to critical bugs from rc1, as well as fix any additional critical bugs found since, until we have no criticals and will then roll out the official 1.0.

To read the full release notes, please see http://community.aegirproject.org/1.0-rc2

Note for upgraders: I screwed up and tagged 1.0-rc2 but we forgot to update the upgrade.sh.txt in Provision so that AEGIR_VERSION points to '6.x-1.0-rc2'. It still reflects the previous release 0.4-rc1.

The upgrade.sh documentation is at http://community.aegirproject.org/node/431 : I have changed the link to upgrade.sh.txt to point to the 6.x-1.x branch version, which contains the correct parameters. It is always best to check the parameters at the top of the script before upgrading anyway, but this is just a heads up.

In short: if you are upgrading, make sure AEGIR_VERSION points to 6.x-1.0-rc2 so that you end up upgrading to the correct version.

Cheers and thanks to all our users!
