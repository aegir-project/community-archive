What is the minimum version of Drush required by Aegir 2.x?

I found two hints, with contradictions:

The Aegir 2.0 release note (http://community.aegirproject.org/2.0) mentions Drush 5.5 as minimum for Aegir 2.x.

On the other side, the install manual (http://community.aegirproject.org/installing/debian) says I can run an automatic install on Wheezy, without any manual step. Hence Debian Wheezy has Drush 5.4 in stable repository, this indicates as Drush 5.4 as minimum for Aegir 2.x.

Which doc is the correct one? Or does it really matter whether Drush 5.4 or Drush 5.5 is the minimum requirement?
